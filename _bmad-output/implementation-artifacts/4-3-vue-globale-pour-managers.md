# Story 4.3: Vue globale pour managers

Status: ready-for-dev

## Story

As a Chef superviseur ou Manager,
I want voir tous les rapports du système selon mon niveau hiérarchique,
So that j'aie une vision complète des données collectées.

## Acceptance Criteria

**Given** je suis un Manager
**When** j'accède au système
**Then** je vois tous les rapports validés par superviseurs
**And** je peux les approuver définitivement ou les rejeter
