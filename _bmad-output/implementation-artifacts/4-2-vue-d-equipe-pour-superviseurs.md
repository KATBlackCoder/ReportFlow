# Story 4.2: Vue d'équipe pour superviseurs

Status: ready-for-dev

## Story

As a superviseur,
I want voir tous les rapports de mon équipe et leurs statuts,
So that je gère efficacement les validations et corrections.

## Acceptance Criteria

**Given** je suis un superviseur authentifié
**When** j'accède à "Équipe"
**Then** je vois tous les rapports de mes subordonnés
**And** je peux valider ou rejeter avec commentaires
