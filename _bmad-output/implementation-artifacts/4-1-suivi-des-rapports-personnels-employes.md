# Story 4.1: Suivi des rapports personnels (Employés)

Status: ready-for-dev

## Story

As a employé,
I want voir la liste de mes rapports soumis avec leur statut,
So that je suive l'évolution de mes soumissions et corrige si nécessaire.

## Acceptance Criteria

**Given** j'ai soumis plusieurs rapports
**When** j'accède à "Mes rapports"
**Then** je vois la liste chronologique avec statuts
**And** je peux voir les commentaires de validation

## Tasks / Subtasks

- [ ] Interface suivi rapports personnels
- [ ] Indicateurs statut visuels
- [ ] Historique soumissions
- [ ] Accès corrections rejetées
