# Story 3.5: Accessibilité WCAG 2.1 AA complète

Status: ready-for-dev

## Story

As a utilisateur en situation de handicap,
I want accéder pleinement à toutes les fonctionnalités,
So that je puisse travailler de manière autonome et efficace.

## Acceptance Criteria

**Given** j'utilise un lecteur d'écran
**When** je navigue dans le questionnaire
**Then** toutes les colonnes et contrôles sont annoncés correctement
**And** je peux saisir des données uniquement au clavier

## Tasks / Subtasks

- [ ] Conformité WCAG 2.1 AA complète
- [ ] Support lecteurs d'écran
- [ ] Navigation clavier totale
- [ ] Tests accessibilité automatisés
